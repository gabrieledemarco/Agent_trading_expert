model_intraday_momentum_trading_with_machine
APPROVED

Architecture

LSTM-based model with dual LSTM layers (128 and 64 units), dropout regularization (0.2), and dense output layers for price movement prediction.

Input Features

  • close_price - Closing price data
  • volume - Trading volume
  • price_returns - Daily price returns
  • moving_averages - 5/10/20 period MAs

Hyperparameters

model = { "lstm_units_1": 128, "lstm_units_2": 64, "dropout_rate": 0.2, "learning_rate": 0.001, "batch_size": 32, "epochs": 100 }

Training Data

  • Symbol: SPY, QQQ
  • Timeframe: 5-minute, 15-minute bars
  • Period: 2023-01 to 2024-12
  • Samples: ~50,000

Performance Metrics

  • Sharpe Ratio: -0.39
  • Win Rate: 47%
  • Max Drawdown: 2.3%
  • Volatility: 2.3%
